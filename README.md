# Campus Issue Management System (ServiceNow)

A custom ServiceNow scoped application designed to automate and streamline the reporting, routing, and resolution of campus-related incidents (WiFi, Maintenance, Electrical).

## 🚀 Key Features & Implementation
* **Custom Data Model:** Built a specialized table tracking 12 unique variables including `Category`, `Priority`, `Status`, and assignment tracking fields.
* **Intelligent Routing Logic:** Configured server-side automation logic (`Auto Route WiFi Issues`) to dynamically assign tickets to the correct support team based on incoming category values.
* **Streamlined UX:** Structured a multi-column form layout optimizing data entry for end-users.
